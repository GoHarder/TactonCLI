# Tacton CLI Tools
> A simple command line program to help with Tacton development

## Requirements
- Node.js v12.16.1 or greater

## Install
```shell
npm link
```

## Uninstall
```shell
npm unlink
```

## Commands
| Command         | Description                |
| --------------- | -------------------------- |
| help            | open the help menu         |
| class-to-domain | convert classes to domains |
